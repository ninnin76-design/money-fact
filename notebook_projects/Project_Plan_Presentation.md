# [기획서] Stock-Dakku: AI 기반 수급 분석 및 투자 다이어리 플랫폼

## 1. 프로젝트 개요 (Executive Summary)

### 1.1 프로젝트 명
- **Stock-Dakku (스톡다꾸)**
- *부제: "투자가 일상이 되는 기록, 수급이 보이는 투자"*

### 1.2 기획 배경 및 문제 정의
- **기존 문제점**: 
    - 대부분의 HTS/MTS는 차갑고 전문가 중심적인 UI로 인해 초보 투자자(주린이)들에게 진입 장벽이 높음.
    - "내가 왜 샀는지", "어떤 감정이었는지" 기록하지 않아 뇌동매매가 반복됨.
    - 재무제표와 수급 데이터가 중요하지만, 숫자가 너무 많아 직관적으로 이해하기 힘듦.
- **해결 방안 (Solution)**:
    - **직관적 수급 시각화**: 연기금, 외국인, 기관의 수급을 게이지 바와 점수로 표현.
    - **다이어리(다꾸) 감성 UI**: 투자를 딱딱한 업무가 아닌, 나만의 다이어리를 꾸미듯 즐거운 경험으로 전환.
    - **초개인화 AI 비서**: 어려운 경제 용어 대신, 친근한 대화체(주치의, 김부장 등)로 투자 조언 제공.

---

## 2. 핵심 기능 (Key Features)

### 2.1 실시간 수급 트래킹 (Supply Tracker)
- **기능 설명**: 주요 세력(연기금, 외국인, 기관)이 최근 5일/10일/20일 동안 연속 순매수한 종목을 실시간으로 포착.
- **차별점**: 단순한 순위 나열이 아닌, 세력별 "수급 파워"를 점수화하여 게이지 바로 시각화.
- **사용자 경험**: "지금 연기금이 쓸어 담고 있는 종목이 뭐지?"라는 질문에 1초 만에 대답.

### 2.2 AI 재무 주치의 (Financial Doctor)
- **기능 설명**: 복잡한 재무 지표(PER, PBR, ROE, 부채비율)를 AI가 분석하여 건강검진 결과처럼 통보.
- **진단 예시**:
    - "현재 PER 8배로 역사적 저평가 상태입니다. 줍줍 구간입니다!"
    - "부채비율이 높으니 단기 대응만 하세요. 혈압 주의!"
- **디자인**: 병원 차트나 처방전 컨셉의 UI 활용.

### 2.3 투자 성향별 포트폴리오 설계
- **기능 설명**: 사용자의 시드머니와 투자 성향(공격형/중립형/방어형)을 입력하면 최적의 포트폴리오 비중 제안.
- **페르소나**: "김부장" 캐릭터가 등장하여 "대표님, 이 종목은 무조건 가야 합니다!" 같은 직장인 컨셉의 유머러스한 조언 제공.

### 2.4 다꾸(다이어리 꾸미기) 스타일 UI/UX
- **디자인 컨셉**: Neon Cyberpunk + Diary Deco
- **폰트**: 'Gaegu(개구쟁이)', 'Outfit' 등 손글씨체와 현대적인 산세리프의 조화.
- **인터랙션**: 스티커 붙이기 효과, 형광펜 밑줄 긋기 애니메이션 등 감성적인 마이크로 인터랙션 적용.

## 3. 기술 스택 및 아키텍처 (Tech Stack)

### 3.1 프론트엔드 (Frontend)
- **HTML5/CSS3**: Vanilla CSS를 활용한 고성능 커스텀 디자인 (Tailwind 지양, 독창적 감성 구현).
- **Lightweight Charts**: 가볍고 빠른 금융 차트 라이브러리 활용.
- **JavaScript (ES6+)**: 별도 프레임워크 없이 모던 JS로 반응형 앱 구현.

### 3.2 백엔드 및 데이터 (Backend)
- **Node.js + Express**: REST API 서버 구축.
- **KIS API (한국투자증권 오픈API)**:
    - 실시간 호가 및 체결 데이터 연동.
    - 기간별 투자자(기관/외국인) 매매 동향 데이터 수집.
    - 재무 비율 및 기업 정보 호출.
- **NotebookLM Integration**: 기획 데이터 및 투자 리포트 생성 연동 (예정).

---

## 4. 기대 효과 및 비즈니스 모델

### 4.1 사용자 기대 효과
- **데이터 기반 의사결정**: 감이 아닌 '수급'과 '실적' 데이터에 근거한 투자 습관 형성.
- **투자 지속성 증대**: 다이어리를 꾸미는 재미 요소로 인해 앱 체류 시간 및 재방문율 증가.

### 4.2 비즈니스 모델 (BM)
- **Freemium 모델**: 기본 수급 데이터 무료, '실시간 AI 심층 분석' 및 '기관 따라잡기 알림'은 유료 구독.
- **디지털 굿즈 판매**: 투자 다이어리 템플릿, 한정판 스티커 팩 판매.

---

## 5. 향후 로드맵 (Roadmap)

1.  **Phase 1 (MVP)**: 수급 랭킹 확인 및 기본 차트, AI 한줄 코멘트 구현 (완료).
2.  **Phase 2 (Mobile App)**: React Native를 이용한 모바일 앱 출시 및 푸시 알림 기능 탑재.
3.  **Phase 3 (Community)**: 사용자끼리 투자 다이어리를 공유하고 자랑하는 '주식스타그램' 커뮤니티 오픈.
