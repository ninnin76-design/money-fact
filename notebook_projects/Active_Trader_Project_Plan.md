제공해주신 기획안 **'Stock-Dakku: Active Trader Edition'**은 단순한 보조지표 제공을 넘어, 시장의 핵심 동력인 **'스마트 머니(Smart Money)'를 추적하는 실전형 주식 비서**를 지향하고 있습니다.

이 기획안의 핵심 내용을 4가지 주요 영역으로 나누어 상세히 분석해 드립니다.

### 1. 킬러 기능: "돈의 흐름" 시각화
이 플랫폼은 투자자가 "진짜 세력"의 유입 여부를 판별할 수 있도록 다음과 같은 차별화된 기능을 제공합니다.
*   **실시간 세력 수급 레이더**: KIS API의 웹소켓을 활용해 장중 **외국인·기관의 잠정 매수치와 프로그램 매매 현황**을 차트 위에 실시간으로 오버레이합니다.
*   **호가창 파워 분석**: 매수/매도 잔량 비율과 체결 강도를 분석하여 게이지 바 형태로 시각화하며, 특히 **체결 강도가 120%를 초과하고 순매수 체결량이 급증**할 때 기술적 매수 신호를 발생시킵니다.
*   **세력 평단가 추적기**: 최근 3일, 5일, 20일간의 데이터를 기반으로 **세력의 평균 단가를 추정**하여 차트에 표시함으로써, 이를 강력한 지지선이나 매수 기회로 활용하게 합니다.
*   **뉴스 감성 분석(AI)**: 실시간 뉴스를 분석해 단순히 나열하는 것이 아니라, 주가에 미칠 영향도를 **1~10점 사이의 'Impact Score'**로 수치화하여 제공합니다.

### 2. 기술 아키텍처: "저지연 및 고성능"
전문 투자자를 타겟으로 하는 만큼, 데이터 처리 속도와 정확성에 최적화된 설계를 제안합니다.
*   **Backend**: 비동기 처리에 강점이 있는 **Node.js**를 선택했으며, 초단기 데이터는 DB 대신 **Redis(In-Memory)**에서 처리하여 지연 시간(Latency)을 최소화합니다.
*   **Hybrid Data 수집**: 핵심 종목은 웹소켓으로 실시간 푸시하고, 나머지 종목은 주기적 폴링(Polling)과 캐싱을 통해 API 호출 제한 내에서 효율적으로 관리합니다.
*   **Frontend**: **React Native**를 통해 모바일 최적화를 구현하고, 금융 전용 라이브러리인 **Lightweight Charts**를 사용하여 대시보드형 UI를 제공합니다.

### 3. 비즈니스 모델 및 수익화
사용자 층을 구분하여 단계별 수익을 창출하는 구조를 가집니다.
*   **구독 모델**: 무료(Basic) 사용자에게는 지연 시세와 기본 차트를 제공하고, **Pro(유료) 사용자**에게는 이 플랫폼의 핵심인 **실시간 수급/호가 데이터 및 세력 평단가 차트**를 제공합니다.
*   **시그널 푸시**: 특정 수급 변곡점이 발생할 때 실시간 알림을 보내는 서비스로, 건당 과금 모델까지 고려하고 있습니다.

### 4. 개발 로드맵
총 6주 내외의 집중 개발 기간을 거쳐 시장에 진입할 계획입니다.
*   **1~2주 차**: KIS API 연동 및 Redis 캐싱 서버 등 **데이터 코어 구축**에 집중합니다.
*   **3~4주 차**: 수급 데이터와 주가를 합성한 **복합 차트 엔진**을 구현합니다.
*   **5~6주 차**: 실제 HTS와 데이터를 비교하여 **정합성을 검증**하는 알파 테스트를 진행한 후 정식 출시합니다.

이 기획안은 **데이터 기반의 객관적 지표**를 원하는 2040 실전 투자자들에게 강력한 도구를 제공하는 것을 목표로 하고 있습니다.
