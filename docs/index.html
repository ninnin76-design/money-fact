🗺️ 섹터별 자금 흐름 히트맵

국내 주요 섹터(업종) 내 핵심 종목들에 유입된 외인/기관의 실제 자금흐름을 바탕으로, 오늘 시장에서 가장 뜨거운 테마와 차가운 테마를 직관적인 온도 색상으로 시각화하여 브리핑합니다.

📡 하이브리드 레이더 (Hybrid Radar)

화면 멈춤 없이 15분마다 정밀 스캔을 백그라운드에서 진행하여 수급 유효 종목만 필터링해 중계합니다.

1. 와이드 스캔: 전 종목 리얼타임 모니터링
2. 후보군 압축: 세력 개입 의심 종목 선별
3. 정밀 분석: 외인/기관 연속성 데이터 산출
4. 하이패스(v4.2.1): 주말/야간 상관 없이 클릭 즉시 실시간 차트 분석
⏱️ 시간 표시의 의미
• [확정] 시간: 서버가 데이터를 성공적으로 분석한 최종 시점입니다. (실패 시에도 마지막 성공 시점이 정직하게 유지됩니다.)
• (확인 완료) 시각: 내 앱이 서버와 통신하여 최신 여부를 체크(Sync)한 시간입니다. 데이터 변화가 없더라도 이 시간은 최신으로 갱신됩니다.

📊 전일 대비 거래량 (면 막대그래프)

차트의 선(테두리)은 전일 거래량을, 채워진 면은 당일 현재까지의 거래량을 나타냅니다. 아직 면이 덜 채워졌다면, 아직 거래가 덜 터졌음을 직관적으로 보여줍니다.

🌡️ 오늘의 시장 종합 심리 (온도계)
