# 💰 머니 팩트(Money Fact) v3.0 : 프리미엄 전략 가이드 📈

**"데이터는 결코 거짓말을 하지 않습니다. 세력의 흔적을 수익으로 바꾸세요."**

머니 팩트는 복잡한 주식 시장에서 외국인과 기관의 실시간 수급을 추적하여 승률 높은 타점을 제시하는 똑똑한 투자 비서입니다. 각 탭의 기능을 완벽하게 숙달하여 수익을 극대화해 보세요!

---

## 🏛️ 1. 대쉬보드 (Dashboard)
**시장의 거대한 돈의 흐름을 한눈에 파악합니다.**

*   **📊 섹터 TOP 6**: 실시간 자금 유입이 가장 활발한 상위 6개 업종을 브리핑합니다. 오늘 어떤 테마가 대장인지 방황할 필요가 없습니다.
*   **📢 실시간 팩트 티커**: 화면 상단에 흐르는 전광판은 단순 정보가 아닙니다. 시장 전체에서 포착되는 [🔥동반쌍끌이]나 [💎평단가 하회] 종목을 실시간으로 중계합니다.
*   **🌙 야간 시세 연동**: 장 마감 후 밤 8시까지 이어지는 넥스트레이드(Nextrade) 거래 대금까지 합산하여 내일의 전략을 미리 세울 수 있습니다.

---

## ⚡ 2. 연속 매매 (Main List)
**1%의 주도주를 찾아내는 머니 팩트의 심장입니다.**

이 탭에서는 설정한 기준에 부합하는 수급 정예 종목들이 실시간으로 나열됩니다.

### 🔋 양음블럭 (수급 에너지)
*   `[🟥🟥🟥🟥]` **4칸**: 외인+기관 동시 대량 매집 (강력 매수 신호)
*   `[🟥🟥⬜⬜]` **1~3칸**: 에너지가 충전 중이거나 한쪽만 주도하는 상태
*   `[🟦🟦🟦🟦]` **파란 블록**: 세력이 대량 이탈 중 (주의/관망 신호)

### 🏷️ 6대 수급 패턴 이름표
| 패턴 | 의미 |
| :--- | :--- |
| **[🔥 동반 쌍끌이]** | 외인/기관 합공! 승률이 가장 높은 베스트 패턴 |
| **[✨ 변곡점 발생]** | 매도에서 매수로 유턴한 추세 반전 지점 |
| **[🤫 히든 매집]** | 주가는 횡보하지만 수급만 몰래 채워지는 구간 |
| **[🏢 기관 주도]** | 국내 기관이 바닥을 지지하며 끌어올리는 중 |
| **[🌎 외인 주도]** | 글로벌 큰손들이 물량을 싹쓸이하는 중 |
| **[❄️ 동반 이탈]** | 모두가 떠나는 자리, 절대 진입 금지 |

## ⭐ 3. 관심 종목 (Watchlist)
**내가 선택한 종목을 24시간 철통 감시합니다.**

*   **➕ 종목 추가/삭제**: 궁금한 종목을 무제한으로 추가하세요. 시장 주도주와 동일한 알고리즘으로 즉시 정밀 분석을 시작합니다.
*   **📊 수급박스 (상세차트)**: 종목을 클릭하면 상단에 외인/기관의 매집/이탈 일수가 바(Bar) 차트로 표시되어 연속성을 한눈에 봅니다.
*   **💎 세력 평단가(VWAP)**: 세력이 물량을 모은 진짜 원가를 계산합니다. 평단가보다 저렴한 구간은 **'안전 마진'** 확보 구간입니다.
*   **🌡️ 투자 심리 온도**: 시장의 뜨거운 관심을 온도로 표시하여 과열 여부를 판단합니다.

## ⚙️ 4. 설정 (Settings)
**나만의 투자 성향에 맞게 비서를 최적화합니다.**

### ☁️ 스마트 데이터 백업
*   **나만의 키(닉네임)**: 고유 닉네임을 설정하고 중복 확인을 하세요.
*   **데이터 복구**: 기기변경 시에도 등록한 관심종목 데이터 완벽 복원 ✅ (닉네임만 기억하세요!)

### 🎯 감지 민감도 개별 설정 (PRO)
알림이 너무 잦거나 둔하다고 느껴지시나요? 직접 조절하세요!
*   **매수 포착 기준**: 며칠 연속 매수 시 신호를 줄지(2~30일) 설정
*   **매도 경고 기준**: 며칠 연속 매도 시 대피 신호를 줄지 설정
*   **매집 포착 기준**: '히든 매집'을 감지할 기간 설정

### 🌙 예약 시스템
장 중에는 데이터 안정성을 위해 설정값이 예약되며, 장 마감 후 시스템에 정밀하게 반영되어 다음 날 최상의 결과를 보여줍니다.
